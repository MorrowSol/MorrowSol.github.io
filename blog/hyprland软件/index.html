      通知

      mako,dunst都可以

      dm

      选择sddm，安装pacman即可

      主题：直接搜sddm theme

      我选的是rating榜第一的 sugar candy https://store.kde.org/s/KDE%20Store/p/1312658

      # 依赖
sudo pacman -S --needed sddm
qt5‑graphicaleffects
qt5‑quickcontrols2
qt5‑svg

      下载安装包，解压到/usr/share/sddm/themes目录下，在conf中启用即可

      sudo tar ‑xzvf ~/Downloads/sugar‑candy.tar.gz ‑C /usr/share/sddm/themes

      测试（预览）主题

      sddm-greeter --test-mode --theme /usr/share/sddm/themes/sugar‑candy

      sddm配置文件有三个地方存放，默认在/usr/lib/sddm/sddm.conf.d

      如果我们改配置的话自己创建新的配置文件

      配置文件优先级，下面的高

      • /usr/lib/sddm/sddm.conf.d
      • /etc/sddm.conf.d
      • /etc/sddm.conf

      我用的最后一个，复制一份配置

      sudo cp /usr/lib/sddm/sddm.conf.d/default.conf /etc/sddm.conf

      更改主题，将theme改为sugar-candy，注意没有空格，主题名和文件夹名对应

      [Theme]
Current=sugar-candy

      重启即可

      我的两个屏幕分别是2560x1440的外接显示器和2880x1800的笔记本内屏，两个屏幕显示均没有问题，只是用户名的字体大小有点小，在主题的Components/Input.qml中 TextField字段 id：username中加入font.pointSize: root.font.pointSize * 1

      网络

      使用Networkmanager即可，安装系统应该安装了

      系统托盘安装：

      sudo pacman -S nm-applet

      配置文件中启用 ：

      exec-once = nm-applet

      终端中ui：nmtui

      命令行就算了

      蓝牙

      安装并启动bluez和bluez-uitls

      sudo pacman -S bluez bluez-utils
sudo systemctl enable bluetooth.service
sudo systemctl start bluetooth.service

      系统托盘安装：

      sudo pacman -S blueman

      配置文件中启用 ：

      exec-once = blueman-applet

      这个终端中的ui：bluetuith不太好用

      输入法

      安装：fcitx5

      sudo pacman -S fcitx5-im fcitx5-chinese-addons fcitx5-pinyin-zhwiki (词库)

      在环境变量中添加：

      sudo nano /etc/environment

GTK_IM_MODULE=fcitx
QT_IM_MODULE=fcitx
XMODIFIERS=@im=fcitx
SDL_IM_MODULE=fcitx
GLFW_IM_MODULE=ibus

      配置文件中启用：

      exec-once = fcitx5 -d

      剪贴板

      安装：cliphist

      sudo pacman -S cliphist

      配置文件中启用：

      exec-once = wl-paste --watch cliphist store

      声音

      linux内核默认有alsa模块

      alsa-firmware包 软件包包含了某些声卡（如创新SB0400 Audigy2）可能需要的固件。

      一些较新型号的笔记本电脑（2019年底/2021）需要sof-firmware包 和 alsa-ucm-conf包，它们使用 Sound Open Firmware 提供的固件实现驱动

      安装后两个：

      sudo pacman -S sof-firmware alsa-ucm-conf

      ui控制声音：

      安装PulseAudio：

      sudo pacman -S pulseaudio

      ui前端：

      sudo pacman -S pavucontrol

      ui调整

      firefox

      wayland现在已经默认开启了，没发现其他bug，这里主要是更改ui主题

      firefox与其他浏览器不同的是可以更改ui设置，不是firefox扩展那的主题，只是颜色改变

      参考 定制指南，应用 userChrome.css，要先在 about:config 开启 toolkit.legacyUserProfileCustomizations.stylesheets。 然后进入 about:support，打开配置文件夹，新建 chrome 目录

      主题store：https://trickypr.github.io/FirefoxCSS-Store.github.io/

      把主题的css文件复制到chrome文件夹下，重启即可
